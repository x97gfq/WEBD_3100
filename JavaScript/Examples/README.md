
# WEBD 1000 — JavaScript Intro (Node.js Examples)

A set of small, **one-concept-per-file** examples designed to be run with **Node.js**.
Each file is heavily commented, prints to the console using `console.log()`, and now includes a
**Starter Exercises** section at the bottom.

## How to run
1. Install Node.js (LTS is fine).
2. Open a terminal in this folder.
3. Run any file, e.g.:
   ```bash
   node 01_variables.js
   ```

### How to use the Starter Exercises
- Scroll to the bottom of each file and look for the **Starter Exercises** block.
- Complete items marked with `// TODO:` directly in the file.
- Re-run the file to see your output change.

## Files
- `01_variables.js` — Variable declaration (`let`/`const`) and assigning many data types; `typeof` demo.
- `02_if_statement.js` — Basic `if / else if / else` with a simple grading example.
- `03_switch_statement.js` — `switch` control flow with `break` and `default`.
- `04_for_loop.js` — Classic `for` loop (counting and accumulating a sum).
- `05_while_loop.js` — `while` loop (countdown), with tips to avoid infinite loops.
- `06_array_of_strings.js` — Creating and using an array of strings.
- `07_random_number.js` — Generating random numbers with `Math.random()` and a helper for an integer range.
- `08_basic_function.js` — Define a function and call it; logging inputs/outputs.
- `09_function_with_args_and_return.js` — Function that takes **arguments** and **returns** a value.
- `10_string_split.js` — Using `String.prototype.split()` to turn a string into an array.

> These are intentionally small so you can run and edit them independently.
