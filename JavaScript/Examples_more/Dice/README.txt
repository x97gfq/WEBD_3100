Dice Roll Game - WEBD 1000 Example

This project demonstrates a simple HTML and JavaScript dice roll game.

Included Files:
- index.html: The main webpage with dice display, play button, and message area.
- script.js: Contains JavaScript code to roll two dice and update the page.

How to Run:
1. Open index.html in any web browser.
2. Click the "Roll Dice" button to roll two dice.
3. The results will appear on the page, along with a message indicating if the dice matched.

JavaScript Overview:
- Generates two random numbers between 1 and 6.
- Updates the DOM with the dice values.
- Displays a message if the dice match.
- Uses an event listener to trigger the dice roll when the button is clicked.
